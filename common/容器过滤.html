    容器过滤

    将数据容器中，数据列表项数小于或大于给定数值的数据容器路径删除。 比如，将最小值设置为1时，就可以过滤掉容器中包含的空的数据列表

    输入端(InputSide)

    名称 描述 数据结构 数据类型
    数据容器 需要过滤的数据容器 容器数据 Object
    数列最小值 数据容器保留的最小数据列项目个数（若数据列表中的项目个数小于该值，则该数据列表将会被从数据容器中删除） 单个数据 Int32
    数列最大值 数据容器保留的最大数据列项目个数（若数据列表中的项目个数大于该值，则该数据列表将会被从数据容器中删除） 单个数据 Int32

    输出端(OutputSide)

    名称 描述 数据结构 数据类型
    数据容器 过滤后的数据容器 容器数据 Object
