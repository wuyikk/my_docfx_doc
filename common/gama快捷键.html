    GAMA快捷键

    画布操作：

    右键 + 点击 = 平移画布

    左键 + 拖动 = 框选

    双击画布空白处 = 弹出卡片搜索框

    Shift + 左键 + 拖动 = 增加框选

    Ctrl + 左键 + 拖动 = 减少框选

    双击鼠标中键 = 显示全部卡片

    菜单栏操作：

    左键单击拖拽进画布 = 放置卡片

    双击一级菜单 = 隐藏二级菜单，再次双击展开二级菜单

    单卡片操作：

    左键 + 拖动 = 拖动选中的对象

    Alt + 左键 + 拖动 = 复制选中的对象

    右键 + 点击 = 对象菜单

    ctrl + alt + 鼠标点击卡片中间 = 查看卡片在菜单中的位置

    连接输入/输出端：

    左键 + 拖动 = 创建一个新连线

    左键 + Shift + 拖动 = 创建新线且不割断旧连线

    左键 + Ctrl + Shift + 拖动 = 将此输入/输出端所有连线都换连至另一个输入/输出端

    右键输出端卡片选择数据连线/左键 + Ctrl + 拖动 = 删除连线

    ctrl + 点击卡片中间区域 = 批量连接出入口

    ctrl + M = 批量连接多卡片

    多卡片操作：

    左键其中一卡片 + 拖动 = 拖动所选个多对象

    右键空白处 = 菜单（卡片打包、关闭预览等）

    文件操作：

    Ctrl + S = 保存文档

    Ctrl + Shift + S = 文档另存为

    编辑

    Ctrl + Z = 撤销

    Ctrl + Y = 重做

    Ctrl + C = 复制

    Ctrl + V = 粘贴

    Delete = 删除

    Ctrl + A = 选取全部

    Ctrl + F = 弹出搜索卡片框
