    创建集合

    创建一个集合，集合中的元素不会重复。（使用默认的重复判断方法，某些特定数据可能不支持默认方法去重）

    输入端(InputSide)

    名称 描述 数据结构 数据类型
    列表 输入一个列表来创建集合，集合是一个不包含重复元素的列表 列表数据 Object

    输出端(OutputSide)

    名称 描述 数据结构 数据类型
    集合 去掉列表中重复的元素后的集合 列表数据 Object
    映射 由输入列表至集合的序号的映射，这个序号的每一位对应着“原列表元素在新集合的第几个”。 列表数据 Int32
